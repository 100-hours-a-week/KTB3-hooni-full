# 1주차 과제 - CLI 프로그램 제작 (동기 프로그램)

## 키오스크 프로그램

### 기능 명세
1. 최소한 기본 메뉴를 선택해야 결제할 수 있다.
   <br>
    1.1. 기본 메뉴는 버거 단품과 버거 세트로 구성된다.
   <br><br>
2. 기본 메뉴를 선택한 후 추가 메뉴를 선택할 수 있다.
   <br>
    2.1. 추가 메뉴는 기본 메뉴를 포함하여 음료와 사이드 메뉴가 있다.
   <br>
    2.2. 추가 메뉴를 선택 후 또 추가 메뉴를 선택할 수 있다.
   <br><br>
3. 매 선택 이후에 결제를 진행할 수 있다.
   <br>
   3.1. 결제를 선택하면 현재까지 선택한 메뉴들의 총합을 화면에 띄운다.
   <br>
    3.2. 사용자에게 금액을 입력받는다.
   <br>
   &nbsp; 3.2.1. 결제하기에 충분한 금액이라면 결제를 진행한다.
<br>
&nbsp; &nbsp; 3.2.1.1. 잔돈이 남는다면 차액을 반환한다.
&nbsp; 3.2.2. 결제 금액이 부족하다면 금액을 재입력받는다.
   <br><br>
4. 결제가 완료되면 주문을 완료하고 첫 화면으로 돌아간다.


5. 메뉴는 메뉴 이름과 금액을 함께 보여준다.


6. 주문을 진행할 때 현재까지 선택된 메뉴들의 금액 총합을 보여준다.
